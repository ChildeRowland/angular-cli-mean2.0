What is this?

This is a Angular2 MEAN stack application.

What is it supposed to do?

Photo Recall is a memory game. Users are shown an image for a brief duration, then answer questions about the content. After sign-up, users can play unlimited sessions. Users can apply to be Game Masters, and build and post their own sessions.

What can it actually do?

Currently, users can sign-up, sign-in, logout, read this FAQ, and see early builds of a game session. Although at this point, I delete the user collection often.
